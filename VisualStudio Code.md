# VisualStudio Code

*VisualStudio Code* - редактор кода для различных языков программирования.

Чтобы установить русский язык необходимо скачать дополнительный плагин из самой программы. Слева на боковой панели находим ярлычок в виде 4-ёх квадратиков. В открывшемся окне в поисковой строке печатаем слово *"Russian"* и устанавливаем плагин.

Отсюда же будем в дальнейшем скачивать расширения для различных языков программирования.
